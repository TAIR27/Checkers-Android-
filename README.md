# Checkers-Android
Android checkers game with full game logic, GUI, and persistent player data
# Checkers Mobile Application

## Features
- Full implementation of checkers rules (legal moves, captures, multi-captures, king promotion)  
- Interactive GUI with multiple screens (game, menu, leaderboard)  
- Background music using Android Services  
- Persistent storage of player data using SharedPreferences  

## Architecture
- Separation between game logic and UI (GameManager and Activities)  
- Modular and maintainable code structure  

## Technologies
- Java  
- Android Studio  

## How to Run
1. Open the project in Android Studio  
2. Run the application on an emulator or device
